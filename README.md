# 都道府県占い

## 概要

本プロジェクトは株式会社ゆめみのコードチェック課題です。

## アプリ仕様

本アプリは 「名前」「誕生日」「血液型」をもとに相性の良い都道府県を占うアプリです。

## 環境

- IDE：Xcode 15.3
- Swift：Swift 5.8
- 開発ターゲット：iOS 16.0
